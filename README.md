# Titanic Survival Prediction
This project builds a machine learning model to predict passenger survival on the Titanic dataset.
The dataset comes from the Kaggle competition:
Titanic - Machine Learning from Disaster.

## Project Goal
Predict whether a passenger survived or not using machine learning.

## Dataset
The dataset contains information about passengers such as:
- Passenger class
- Age
- Sex
- Fare
- Number of relatives onboard

Files used:
- train.csv
- test.csv

## Workflow
1. Data loading
2. Data cleaning
3. Exploratory data analysis
4. Feature engineering
5. Model training
6. Prediction

## Feature Engineering
Additional features created:
- FamilySize
- IsAlone
- Title extracted from passenger name

## Model
Random Forest Classifier was used to train the model.

## Result
Kaggle Score: 0.74162
